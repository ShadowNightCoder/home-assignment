# Assignment 3 - Backend

This project contains a **Node.js server** and **SQLite database** for the SQL Injection assignment.

## How to Run the Project

### 1. Install dependencies
Run:

```bash
npm install
```

### 2. Initialize the database
This will create the `database.sqlite` file:

```bash
node init_db.js
```

### 3. Start the server
Run:

```bash
node server.js
```

## Notes
- Make sure you have **Node.js** installed on your machine.
- The database file will be generated automatically after running the initialization script.
- The server will run locally unless configured otherwise.
